Proyecto: Web Scraping con Selenium.
Elección del tema:
Elegi el tema pues me pareció interesante el término de "Scraping" y realice la búsqueda y note que tenía una particularidad incluso legal del cumplir 
con las condiciones que promueve el creador de donde sacamos la información. Pues realizar Scraping es extraer información de sitios web, además de que
hay muchos programas diferentes que pueden ayudarnos a realizar esto.

¿Por qué me resultó interesante?
Quería saber como se podia implementar este termino de distintas formas, ya que investigando se podia usar para realizar estadisticas sobre el aumento
de precios de un producto de una pagina web, calculos estadisticos en tiempo real, etcetera.

¿Qué aprendió de este?
Aprendi sobre un nuevo metodo para obtener informacion de un sitio web, distintos modos de código que pueden ser utilizados para realizar Scraping, además
de que me sorprendí pues el video que consulte para realizar web scraping dice que en realidad, esto es lo más básico que se hace con Selenium.
